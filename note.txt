# MongoDB Connection in PHP

## Steps to Connect MongoDB with PHP on Windows

1. **Install MongoDB**
   - Download and install MongoDB from [MongoDB Download Center](https://www.mongodb.com/try/download/community).
   - Start the MongoDB server using:
     ```sh
     mongod
     ```

2. **Install XAMPP**
   - Download and install XAMPP from [Apache Friends](https://www.apachefriends.org/).
   - Start Apache and MySQL from the XAMPP control panel.

3. **Install MongoDB PHP Driver**
   - Open Command Prompt and run:
     ```sh
     pecl install mongodb
     ```
   - Enable the extension in `php.ini` by adding:
     ```ini
     extension=mongodb
     ```

4. **Create a PHP Project and Install Dependencies**
   - Navigate to your project directory and run:
     ```sh
     composer require mongodb/mongodb
     ```

5. **Write PHP Code to Connect to MongoDB**
   ```php
   <?php
   require 'vendor/autoload.php';
   
   use MongoDB\Client;
   
   $client = new Client("mongodb://localhost:27017");
   $db = $client->user;
   $collection = $db->users;
   
   $collection->insertOne(["name" => "John Doe", "age" => 25, "email" => "john@example.com"]);
   
   echo "User added successfully!";
   ?>
   ```

# MySQL Connection in PHP using XAMPP

## Steps to Connect MySQL with PHP on Windows

1. **Install XAMPP**
   - Download and install XAMPP from [Apache Friends](https://www.apachefriends.org/).
   - Start Apache and MySQL from the XAMPP control panel.

2. **Create a Database in phpMyAdmin**
   - Open [http://localhost/phpmyadmin](http://localhost/phpmyadmin) in a browser.
   - Click on "New" and create a database named `user`.
   - Create a table `users` with columns `id`, `name`, `age`, `email`.

3. **Write PHP Code to Connect to MySQL**
   ```php
   <?php
   $conn = new mysqli("localhost", "root", "", "user");

   if ($conn->connect_error) {
       die("Connection failed: " . $conn->connect_error);
   }
   
   $sql = "INSERT INTO users (name, age, email) VALUES ('John Doe', 25, 'john@example.com')";
   if ($conn->query($sql) === TRUE) {
       echo "User added successfully!";
   } else {
       echo "Error: " . $conn->error;
   }
   
   $conn->close();
   ?>
   ```
